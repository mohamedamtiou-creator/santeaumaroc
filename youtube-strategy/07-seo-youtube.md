# SEO YouTube — Méthode réutilisable · SantéAuMaroc

> Ce fichier donne le **système** SEO à appliquer à chaque vidéo. Les blocs SEO complets des 50 premières vidéos sont dans les fichiers `06-scripts-*.md` ; les mots-clés des 300 idées sont dans `05-300-idees-videos.md`.

## 1. Recherche de mots-clés (avant de tourner)
- Sources : **barre de recherche YouTube** (autocomplétion), **Google Trends – région Maroc**, « autres recherches » Google, la base **Q/R + blog** du site (déjà riche en requêtes patients réelles), TubeBuddy/VidIQ (option).
- Chercher en **français ET en arabe/darija** (ex. « ضغط الدم », « السكري »).
- Cibler l'**intention** : informationnelle (« c'est quoi / symptômes de ») → gros volume ; transactionnelle (« trouver médecin / rendez-vous ») → conversion.
- Prioriser les requêtes où l'on peut **relier à un médecin** (donc à un RDV).

## 2. Titre SEO
- **≤ 60-70 caractères**, mot-clé principal **au début**.
- Ajouter un **déclencheur de clic** : chiffre, question, bénéfice, « au Maroc », urgence douce.
- Éviter le clickbait mensonger (YMYL = confiance). Exemples : *« Diabète : 7 signes à ne jamais ignorer (au Maroc) »*, *« Comment prendre rendez-vous médical en ligne au Maroc »*.

## 3. Description SEO (structure type)
```
[Ligne 1-2 = accroche + mot-clé principal, visible avant "plus"]
[Paragraphe 1 : résumé du contenu + mots-clés secondaires, 150-250 mots]

👉 Trouvez un médecin et prenez rendez-vous : https://santeaumaroc.com/?utm_source=youtube&utm_medium=video&utm_campaign=<slug>

⏱️ Chapitres :
00:00 …
[timecodes]

🔗 Liens utiles : [page spécialité/ville pertinente]
📌 Playlist : [lien]

⚠️ Cette vidéo est informative et ne remplace pas une consultation médicale.

#hashtag1 #hashtag2 #hashtag3
```
- Placer le **lien site en 1ʳᵉ position** cliquable + **UTM** par vidéo.
- Renvoyer vers la **page la plus pertinente** (spécialité/ville), pas seulement l'accueil.

## 4. Tags (10-15)
Mot-clé principal, variantes, synonymes, versions arabes, nom de marque « SantéAuMaroc », spécialité + ville quand pertinent. Les tags pèsent peu aujourd'hui mais aident sur les fautes/variantes et l'arabe.

## 5. Hashtags (3-5)
3 max affichés au-dessus du titre. Ex. `#SantéAuMaroc #SantéMaroc #Médecin`. Ajouter un hashtag thématique (`#Diabète`, `#Grossesse`).

## 6. Chapitres YouTube
- Toujours commencer à **00:00**, ≥ 3 chapitres, libellés avec mots-clés. Améliore rétention, référencement et « key moments » Google.

## 7. Miniature (règle de marque)
- Format 1280×720, **texte 3-5 mots** très contrasté, visage/émotion ou picto clair, couleurs charte (bleu/émeraude/ambre), logo discret constant.
- **A/B test** systématique (fonctionnalité test miniatures YouTube).
- Cohérence visuelle par série (même gabarit) pour la reconnaissance.

## 8. Signaux de rétention (le vrai moteur de l'algorithme)
- **Hook < 10 s** obligatoire (promesse + enjeu), pas d'intro longue.
- Annoncer tôt le « payoff » de fin (le CTA site).
- Rythme : coupe visuelle toutes les 3-5 s, sous-titres incrustés (conso mobile Maroc majoritaire).
- **Sous-titres FR + AR** = accessibilité + SEO + marché arabophone.
- Écran de fin + cartes vers vidéo/​playlist suivante pour la **session watch time**.

## 9. Localisation arabe (levier #1 de différenciation)
- Ajouter **titre + description + sous-titres traduits (métadonnées localisées YouTube)** sur les vidéos performantes.
- Envisager un **2ᵉ canal / flux dédié arabe** une fois le rythme FR stable.
- Prioriser en arabe : produit (trouver médecin/RDV), maladies chroniques, santé femme & enfant.

## 10. Checklist publication (à cocher pour chaque vidéo)
- [ ] Titre ≤ 70 car, mot-clé au début
- [ ] Miniature testée, texte lisible mobile
- [ ] Description + lien UTM + page profonde pertinente
- [ ] Chapitres depuis 00:00
- [ ] Tags FR + AR + marque
- [ ] 3-5 hashtags
- [ ] Sous-titres FR (+ AR si possible)
- [ ] Écran de fin + cartes
- [ ] Ajoutée à ≥ 1 playlist
- [ ] Commentaire épinglé avec CTA + lien
- [ ] Disclaimer YMYL présent
- [ ] Déclinaison Shorts + article blog programmés
- [ ] Métadonnées localisées AR (si applicable)
